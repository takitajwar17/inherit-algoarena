## 🌟 Mission

**Inherit** is an open-source platform working to bridge the digital divide by making quality coding education more accessible to underserved communities. Our project aims to tackle educational inequality by providing learning experiences that adapt to diverse learning styles, socioeconomic backgrounds, and accessibility needs.

### 🎯 The Problem We're Addressing

* According to the UN, about 2.9 billion people globally still lack internet access
* Coding education resources often remain inaccessible to many communities
* Conventional learning approaches may not accommodate different learning styles or disabilities
* Many high-quality educational resources are behind paywalls

## ✨ Current Features

### 🔄 Learning Platform
* **Interactive Dashboard**: Track learning progress, active quests, and roadmaps
* **Code Playground**: Built-in code editor supporting multiple programming languages (JavaScript, TypeScript, Python, Java, C#, C++, PHP)
* **Learning Roadmaps**: Structured learning paths for various programming topics
* **Quest System**: Gamified learning challenges to build coding skills
* **Accessibility Tools**: Screen reader support and accessibility settings

### 💡 User Experience
* **Responsive Design**: Works across desktop and mobile devices
* **Voice Commands**: Navigate the platform using voice inputs
* **Personalized Learning**: Track progress and pick up where you left off
* **Developer Discussions**: Community area for asking questions and sharing knowledge

### 🌐 Authentication & User Management
* **Secure Sign-up/Sign-in**: User authentication powered by Clerk
* **User Profiles**: Personalized user experience and progress tracking

## 🛠️ Technology Stack

- **Framework**: Next.js (React)
- **Authentication**: Clerk
- **Styling**: Tailwind CSS
- **Analytics**: Vercel Analytics and Speed Insights
- **UI Components**: Custom components with accessibility focus
- **Icons**: React Icons
- **Progress Visualization**: react-circular-progressbar
- **Notifications**: react-toastify

## 🚀 Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn
- Git

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/takitajwar17/inherit.git
cd inherit
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Configure environment variables**
Create a `.env.local` file with necessary credentials (see `.env.example`)

4. **Run the development server**
```bash
npm run dev
# or
yarn dev
```

5. **Build for production**
```bash
npm run build
npm start
```
